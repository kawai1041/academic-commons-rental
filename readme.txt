アカデミックコモンズ貸出用ソフト　使用方法

起動
　kashidashi.exeをダブルクリックする

終了
　画面の右上の×をクリックする。

貸し出し操作(カーソル位置確認要）
　(a)学生証のバーコート読み込み
　(b)貸出物品のバーコード読み込み
　(c)連絡先電話番号の入力（キーボードから、半角数字、ハイフンは無くても良い、Enter必要）
　(d)返却予定時刻の入力（キーボードから、半角数字、コロンは無くても良い、Enter必要）
  (a)-(d)は順不同

　この時点で学生氏名が表示される、氏名が？？？？？？になっているときは在籍していない学生の可能性がある。
　　　口頭で在籍を確認すること
　
　(e)貸出ボタンのクリックまたは貸し出し返却バーコード読み込み

　複数物品を貸し出すときは一点目の貸出後、貸出物品のみバーコードを読ませて、貸出ボタン（またはバーコード）

返却操作(カーソル位置確認要）
　貸出物品のバーコード読み込み
　　　学生番号、物品名、返却予定時刻、学生氏名　が表示される
　返却ボタンのクリックまたは貸し出し返却バーコード読み込み


機器不調時の機器交換
　貸出物品のバーコード読み込み
　機器交換ボタンを押す
　新たに貸し出す機器のバーコードを読む、
　貸出ボタンのクリックまたは貸し出し返却バーコード読み込み


返却時間超過学生への連絡
　返却予定時刻を過ぎている学生に連絡するときは
　current_年月日.csvのファイルを開く（ダブルクリックでexcelで開く）
　現在貸し出し中のデータが表示される。 電話番号の最初の０が消えるので注意必要

　注意：このファイルを開いたまま貸出・返却処理を行うと、でエラーが発生し、貸出ソフトが終了してしまう。
　　　　ファイルを閉じて、貸出ソフトを再起動し、貸出・返却処理を再度行うこと

返却済みデータ
　返却がすんだデータは kashidashi_年月日.csv に記録される。

　注意：このファイルを開いたまま返却処理を行うと、でエラーが発生し、貸出ソフトが終了してしまう。
　　　　ファイルを閉じて、貸出ソフトを再起動し、返却処理を再度行うこと
